Daily Programmer #296

https://www.reddit.com/r/dailyprogrammer/comments/5go843/20161205_challenge_294_easy_rack_management_1/#header

Given a set of 7 letter tiles and a word, determine whether you can make the given word using the given tiles.
Bonuses include using wildcard tiles and scoring, along with finding the longest word you could generate
given the tiles you have.
